# 🚀 Инструкция по деплою (v3.0 - Standalone)

Бот работает **автономно**, без веб-сервера и доменов.
Конфигурации и QR-коды приходят **прямо в чат** администратору.

## 🛠️ Шаг 1: Сборка (На Windows)
1.  Запустите `pack.bat` (в корне проекта).
2.  Получите файл **`release.zip`**.

## ☁️ Шаг 2: Загрузка
Загрузите **только** файл `release.zip` на сервер в папку (например, `/root/` или `~`).

## ⚙️ Шаг 3: Установка
Подключитесь по SSH и выполните:

```bash
# 1. Распаковка
sudo apt install -y unzip
unzip -o release.zip -d wgtg_temp

# 2. Установка
cd wgtg_temp
chmod +x deploy/setup.sh
./deploy/setup.sh
```

**Скрипт запросит:**
- Токен бота
- ID админов
- URL и Пароль от вашей панели WG-Easy.

**Никаких настроек Nginx, SSL или доменов больше не требуется.**
Бот просто запустится и будет работать.

## 🔄 Шаг 4: Обновление
Для обновления просто загрузите новый `release.zip` и выполните:

```bash
unzip -o release.zip -d wgtg_temp
cd wgtg_temp
chmod +x deploy/update.sh
./deploy/update.sh
```
